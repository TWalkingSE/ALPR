# Fixtures versionadas

Use esta pasta para manter amostras rotuladas de regressao do ALPR 2.0.

Estrutura sugerida:

- `images/`: imagens individuais de placa
- `videos/`: videos curtos para consolidacao temporal
- `manifest.template.json`: exemplo de manifesto versionado

Cada fixture deve ter um `id`, o caminho relativo do arquivo e a placa esperada para a avaliacao.
